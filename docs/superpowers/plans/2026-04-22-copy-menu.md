# Copy Menu Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在配置页加入一个小复制按钮，把当前菜单按“每行一项”的格式复制到剪贴板，并可直接粘贴进现有导入列表。

**Architecture:** 继续沿用 `whatToEat/wheel.html` 的单文件结构，在配置区标题行增加一个小按钮；新增一个将 `state.items` 转为导入文本的辅助函数，并在点击按钮时调用 `navigator.clipboard.writeText`。失败与成功反馈复用现有状态文本，避免增加新的 UI 容器。

**Tech Stack:** HTML, CSS, vanilla JavaScript, Node.js assert/vm 测试

- [ ] **Step 3: 实现点击复制逻辑与状态反馈**

```js
async function copyItemsToClipboard() {
  if (!state.items.length) {
    setStatus("当前没有菜单内容，暂时无法复制。");
    return;
  }

  const text = itemsToEditorText(state.items);

  try {
    await navigator.clipboard.writeText(text);
    setStatus(`已复制 ${state.items.length} 项，可直接粘贴到导入列表。`);
  } catch (error) {
    setStatus("复制失败，请检查浏览器是否允许访问剪贴板。");
  }
}
```
